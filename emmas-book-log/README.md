# Emma's Book Log
By Emma Bowen

## Description
I developed this Python script to track and create visualizations regarding the books I've read. This script initially creates a new Pandas DataFrame with book information as the column headers, then loads in the existing newly created CSV file, asks the user to input information regarding the book, such as `Author`, `Title`, `Pages` etc., confirms the information is correct, and saves the information as a new row of data into the DataFrame.   
As I continue to read more books in 2026, I plan to come back to this CSV file and begin to conduct a data analysis in a Jupyter Notebook and create visualizations to highlight my reading trends.

## Data information
- The data file, *book-log-data.csv* was created by me and contains information regarding the books I've read, such as `Title`, `Author`, `Month_Read`, and `Personal_Rating`.
